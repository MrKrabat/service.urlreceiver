# URLReceiver plugin for Kodi

URLReceiver - Send URLs for playback directly on your Kodi device!
Service Addon for KODI (XBMC) v16 Jarvis and newer.

Required to have a standalone client for sending URLs.
Optionally it is offering a web interface.

Website: https://dev.mrkrabat.de/urlreceiver/

Git Repo: https://github.com/MrKrabat/service.urlreceiver

URL Sender: https://github.com/MrKrabat/URLSender

Forum Posting: --
***

Sending URLs with:
- [x] Webinterface http://<KODI_IP>:<PORT>/
- [x] GUI: URLSender - 3 different
- [x] CLI: URLSender - 3 different
- [x] Kodi video plugin support
- [x] Support for URLResolver plugin (YouTube, Twitch, Streamcloud, MP4Upload...)
- [x] Support to directly play video, music and playlist files (mp4, mp3 and m3u8 as example)
- [x] Support for cookies / user agents append "|Cookie=A=B;C=D&User-Agent=Mozilla" (URL encode!)
- [x] Easy to write browser extensions http://<KODI_IP>:<PORT>/urlsend=http://video.url
